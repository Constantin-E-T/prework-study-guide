HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

CSS

  • A margin : indicates how much space we want around the outside of an element.
  • A padding : indicates how much space we want around the content inside an element.
  • A border : A shorthand property for border-width, border-style and border-color
  • A color : Sets the color of text
  • A line-height : Sets the line height

Git

  • Prepared new branch steps: ↴
  • git --version : show you what git version are you using
  • git status : checks what branch we are currently on
  • git checkout main : switch to the main branch
  • git pull origin main : pull down the latest main branch
  • git checkout -b 'branch-name' : creates a new branch and switches to it
  • git status : checks what branch we are currently on
  • Push the new branch steps: ↴
  • git add -A : We use the add -A command to commit all files in the working branch.
  • git commit -m 'your message' : We use the -m flag to associate a message with our commit. Commit messages should be short descriptive messages of the changes implemented
